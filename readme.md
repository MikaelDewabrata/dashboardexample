# 📊 Warung Mie Ayam Bojot – Sales Dashboard

A simple sales dashboard project simulating a real-world **UMKM (micro/small business)** using Looker Studio.

---

## 🔗 Live Dashboard
👉 https://lookerstudio.google.com/reporting/db3975c7-41e2-4759-bf5d-b4c48c006571

---

## 📌 Overview

This project demonstrates how small businesses can use basic data tracking to:

- Monitor revenue and profit  
- Understand product performance  
- Identify trends over time  

---

## 📂 Dataset

This project uses **synthetic (dummy) data** generated using Python.

### Products:
- Mie Ayam  
- Mie Bakso  
- Es Teler  
- Es Teh Manis  

---

## 🧱 Data Structure

| Column   | Description |
|----------|------------|
| year     | Year |
| month    | Month (1–12) |
| date     | First day of month |
| menu     | Product name |
| qty      | Quantity sold |
| revenue  | Total sales (IDR) |
| cogs     | Cost of goods sold |
| profit   | Revenue - COGS |

---

## ⭐ Why This Project Matters

Most UMKM businesses don’t use structured data.

This project shows:
> Even simple data tracking can improve decision-making.
